# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-17

### Added
- Complete project boilerplate with Next.js 14 App Router.
- Authentication integration using Supabase SSR cookies (`@supabase/ssr`).
- Subscription billing integration via Stripe Checkout sessions.
- Stripe Webhook handler validating signatures for customer subscription updates.
- Tailwind CSS v4 & shadcn/ui custom styling components.
- Initial migrations for `subscriptions` table inside Supabase database.
- Comprehensive technical and operation documentation suite under `docs/`.
